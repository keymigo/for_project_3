# Проект по поиску интересующих вакансий с hh.ru

## Цель проекта
Получить вакансии по выбранным профессиям с сайта hh.ru, определить географическое распределение работодателей в Москве и визуализировать их расположение на карте.

## Задачи проекта
- Выбрать 5 несмежных профессиональных направлений.
- Получить вакансии по выбранным профессиям в городе Москва.
- Извлечь и обработать координаты местоположения компаний.
- Визуализировать вакансии на карте.

## Методология и инструменты
Проект выполнен на языке **Python** с использованием следующих библиотек:
- **pandas**, **numpy** — обработка и анализ данных;
- **geopandas** — работа с географическими данными;
- **matplotlib**, **seaborn** — статические визуализации;
- **plotly** — интерактивная визуализация на карте.

Формат проекта — **Jupyter Notebook (`.ipynb`)**.

## Источники данных
Данные о вакансиях получены путём парсинга сайта **hh.ru** по открытым вакансиям, опубликованным в Москве.

## Ход работы
1. Определение списка из 5 интересующих профессий, не являющихся смежными.
2. Сбор вакансий по каждой профессии.
3. Очистка и подготовка данных.
4. Извлечение географических координат компаний.
5. Построение карты с отображением расположения вакансий.

## Визуализация данных
Результаты представлены в виде:
- карты Москвы с отмеченными точками расположения компаний;
- интерактивной визуализации, позволяющей изучить распределение вакансий по городу.

## Результаты и выводы
Проект позволяет наглядно оценить пространственное распределение вакансий по разным профессиям в Москве и может быть использован для:
- анализа концентрации работодателей;
- выбора района для поиска работы;
- первичного исследования рынка труда по выбранным направлениям.

## Ограничения
- Анализ основан только на открытых вакансиях hh.ru.
- Координаты зависят от корректности данных, указанных работодателями.
- Проект не учитывает динамику вакансий во времени.
